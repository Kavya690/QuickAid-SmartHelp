# QuickAid-SmartHelp
QuickAid is a smart emergency web app that provides instant step-by-step guidance and quick access to emergency contacts, and live location support - designed to help users to act fast in critical  situations.

🚑 QuickAid – Smart Emergency Help Web App

📌 Problem Statement

In emergency situations, people often panic and don’t know what to do. They may not remember emergency numbers or correct first-aid steps, which can lead to serious consequences.

💡 Solution Overview

QuickAid is a smart emergency web app that provides instant step-by-step guidance, quick access to emergency contacts, and live location support to help users act fast in critical situations.

✨ Key Features

- 🚨 One-tap SOS emergency button
- 📖 Step-by-step first aid instructions
- 📞 Quick access to emergency contacts
- 📍 Live location sharing
- ⚡ Simple and fast user interface
- 📶 Works offline (basic features)

🛠 Tech Stack Used

- HTML
- CSS
- JavaScript

▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open "index.html" in your browser

🚀 Future Improvements
